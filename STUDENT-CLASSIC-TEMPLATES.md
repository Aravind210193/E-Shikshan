## ğŸ“‹ Student Resume Best Practices

### âœ… DO's:

1. **Education First**
   - Place education section at the top
   - Include GPA if above 3.0 (or percentage if above 70%)
   - List relevant coursework

2. **Projects Matter**
   - Describe 2-4 substantial projects
   - Include tech stack used
   - Mention your specific role
   - Add GitHub links

3. **Skills Section**
   - Categorize skills (Technical, Soft Skills)
   - Be honest about proficiency levels
   - Include both hard and soft skills

4. **Quantify When Possible**
   - "Built app with 100+ downloads"
   - "Led team of 5 members"
   - "Scored 95% in Data Structures"

5. **Keep It One Page**
   - Students should stick to 1 page
   - Use space efficiently
   - Remove irrelevant details

### âŒ DON'Ts:

1. **Don't Lie or Exaggerate**
   - Be truthful about skills
   - Don't inflate GPA or achievements
   - You'll be caught in interviews

2. **Don't Include Irrelevant Info**
   - Skip elementary school details
   - Omit unrelated hobbies
   - No "Date of Birth" or "Father's Name" (unless specifically required)

3. **Don't Use Fancy Fonts**
   - Stick to professional fonts (Arial, Calibri, Times New Roman)
   - Avoid Comic Sans, cursive fonts
   - Keep font size between 10-12pt

4. **Don't Overload with Keywords**
   - Use keywords naturally
   - Don't stuff skills section
   - Quality over quantity

5. **Don't Forget Contact Info**
   - Include professional email
   - Add LinkedIn profile
   - Ensure phone number is correct
